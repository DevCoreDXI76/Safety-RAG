```
📋 정보통신공사 안전서류 초안 생성기 (베타)

현직 20년차 PM이 직접 겪은 불편함이라 만들어봤습니다. 현장 정보 몇 줄만 입력하면 AI가 초안을 잡아드려요.

⚠️ 아직 베타 단계라, 생성된 초안은 반드시 안전관리자님이 직접 검토·수정하신 후 사용해주세요.

편하게 써보시고 어떤 점이 불편한지, 어떤 부분이 쓸만한지 솔직한 의견 주시면 정말 큰 도움이 됩니다 🙏
```

## 3. 사용법 안내 문구

```
📱 사용 방법

1. 텔레그램 앱이 필요해요 (없으시면 앱스토어/플레이스토어에서 "Telegram" 검색 후 설치)
2. 아래 링크 클릭 → 텔레그램 안에서 자동으로 화면이 열립니다
3. 현장명 입력 → 문서 종류 선택 → (TBM일지인 경우) 연동할 위험성평가 회차 선택 → 작업 정보 입력
4. "초안 생성" 버튼 클릭 (최대 30초 정도 걸릴 수 있어요)
5. 생성된 초안은 참고용입니다 — 최종 검토·승인은 안전관리자님이 직접 해주세요

⏱️ 베타 테스트 기간이라 1인당 하루 5회로 생성 횟수가 제한되어 있어요. 더 필요하시면 편하게 말씀해주세요.

👉 링크: https://t.me/<봇 username>/app
```

## 4. 배포(공유) 우선순위

1인개발 10원칙 중 "Do things that don't scale" 원칙과, 20년간 쌓은 정보통신공사/안전관리 분야 네트워크를 활용하는 방향으로 우선순위를 정함.

1. **직접 아는 현장소장/안전관리자/PM 지인 10~15명** — 1:1 개인 메시지로 부탁. 가장 정확한 피드백을 받을 수 있는 그룹.
2. **정보통신공사 관련 협회/카페/밴드** — 업계 특화 커뮤니티.
3. **블라인드 "직접 홍보" 채널** — 업계 재직 이력 기반으로 관련자 노출 가능.
4. **카카오톡 오픈채팅방** — "정보통신공사", "안전관리자" 관련 오픈채팅방 탐색.
5. **디스콰이엇** — 잠재고객보다는 1인개발자 대상 기술/제품 피드백용.

지인 그룹 반응을 먼저 확인하고 이슈(사용성, 버그, 항목 누락 요청 등)를 반영한 뒤, 더 넓은 채널로 확장하는 순서 권장.
